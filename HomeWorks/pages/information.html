Time

Асинхронность

Асинхронность — функция вызывается через время, мы не знаем, когда она отработает.


Двойная асинхронность — используется onclick.


Асинхронная функция — фунция, которая начнёт своё выполнение кода на заднем фоне.

Синхронность

Синхронность — функция вызывается мгновенно без задержек.


Синхронная функция — фунция, которая начнёт своё выполнение кода сразу.

setTimeout ( function, time )

setTimeout — позволяет вызвать функцию один раз через определённый интервал времени.


function — функция, которая вызовется один раз.


time — интервал времени, через который вызоветься функция. (1000 = 1 sec)

setInterval ( function, time )

setInterval — позволяет вызывать функцию регулярно, повторяя вызов через определённый интервал времени.


function — функция, которая будет вызываться регулярно.


time — интервал времени, через который будет вызываться функция. (1000 = 1 sec)

clearTimeout ( timer )

clearTimeout — позволяет остановить таймер.


timer — таймер, который нужно остановить.

clearInterval ( timer )

clearInterval — позволяет остановить таймер.


timer — таймер, который нужно остановить.

class List

add ( class )

add — метод, используемый для добавления класса к элементу.


class — строка с именем класса, которая добавится.

remove ( class )

remove — метод, используемый для удаления класса у элемента.


class — строка с именем класса, которая удалится.

replace ( class )

replace — метод, используемый для замены класса у элемента.


class — строка с именем класса, которая заменится.

contains ( class )

contains — метод, используемый для проверки наличия класса у элемента.


class — строка с именем класса, наличие которого нужно проверить.
